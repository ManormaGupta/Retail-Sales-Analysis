# Retail-Sales-Analysis project

## 📌 Project Overview
This project focuses on managing and analyzing retail sales records. It demonstrates how to handle large datasets, perform data cleaning, and extract meaningful business insights using *Excel* for initial data management and *Python* for advanced analysis.

## 🛠️ Tech Stack Used
* *Data Management:* Microsoft Excel
* *Programming Language:* Python
* *Libraries:* pandas, openpyxl
* *IDE:* VS Code

## 🗂️ Repository Structure
* retail_sales_data.xlsx: Contains the raw and structured retail sales data, including product details, pricing, and transactions.
* retail_analysis.py: Python script used to clean the dataset, handle missing values, and perform exploratory data analysis (EDA) using Pandas.

## 📊 Analysis Focus
The project analyzes key retail metrics such as:
* Sales performance across different product categories
* Monthly and daily sales trends
* Customer purchasing patterns
* Profit margins and revenue generation
